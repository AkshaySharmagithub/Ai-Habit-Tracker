# Full-Stack AI-Powered Habit Tracker App

A modern MERN stack habit tracker application with AI-powered productivity insights using Gemini API.

## Features
- User Authentication
- Habit Tracking
- AI Habit Suggestions
- Progress Dashboard
- Light/Dark Mode
- Responsive UI

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Gemini API
