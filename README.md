# 🚀 AI Habit Tracker

Live Link https://ai-habit-tracker-mu.vercel.app/

A modern full-stack AI-powered habit tracker application built with the MERN stack.

## ✨ Features

* 🔐 User Authentication
* 📅 Daily Habit Tracking
* 🤖 AI-Based Habit Suggestions
* 📊 Progress Dashboard
* 🌙 Light/Dark Mode
* 📈 Weekly & Monthly Analytics
* 📱 Fully Responsive UI

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB

### AI Integration

* Gemini API

## ⚙️ Environment Variables

### Backend `.env`

```env
MONGO_URI=your_mongodb_uri

```

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/AkshaySharmagithub/Ai-Habit-Tracker.git
```

---

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---
